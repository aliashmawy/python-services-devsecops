 Financial Forecasting API (FastAPI on Hugging Face Spaces)

This project provides a financial forecasting API built using **FastAPI** and deployed on **Hugging Face Spaces**.
It leverages Facebook's **Prophet** model to forecast quarterly financials — including **Revenue**, **Expenses**, and **Cash Flow** — for a given company (We should input the company ID for testing for Example you can test with this id "CMP_001").

POST Endpoint – for forecasting:
https://dina-tolba-prophet-model-forecast.hf.space/forecast


GET Endpoint – for API documentation:
https://dina-tolba-prophet-model-forecast.hf.space/docs
